0.0.1 This is the project website for the paper "Economic inequality and its influence on the justification of legitimate income differences.

Last update: martes 08 septiembre 2020 20:07:00

0.0.2 Authors:

  • Julio Iturra, Centro de Estudios de Conflicto y Cohesión Social - COES julioiturra@uchile.cl
  • Juan Carlos Castillo, Departamento de Sociología, Universidad de Chile.
  • Catalina Rufs, Pontificia Universidad Católica de Chile.

Based on the study “Political and Social Beliefs” conducted in 2014 using a representative sample of the Metropolitan Region of Chile, we replicate the original experimental study conducted by Trump (2018). We tested how percieved economic inequality affects afects the global and ocupational based salary gaps in a highly unequal context.

0.0.3 Structure of the project website:

  • The tabs follow the logic of the IPO protocol for academic workflows, based on three main components = Input -> Production -> Output. Here we show only one the main parts of each component: Data \(\rightarrow\) Analysis \(\rightarrow\) Paper.

  • The complete reproducible files are in the Github repo: https://jciturras.github.io/desigualdad_recom/

    • Data: pre-processing and general descriptives
    • Analysis: Balance, Treatment effect analysis (OLS) and annexes
    • Paper: final output
